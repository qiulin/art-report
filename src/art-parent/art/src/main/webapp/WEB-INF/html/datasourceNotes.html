• If using Oracle, please note that ART doesn't come with the Oracle JDBC driver. The driver (ojdbc6.jar or ojdbc7.jar) needs to be manually downloaded and copied to the WEB-INF\lib directory. The application will then need to be restarted. The driver can be downloaded from http://www.oracle.com/technetwork/database/features/jdbc/index-091264.html
  • The SQL Server (Microsoft Driver) is also not included with ART. It can be downloaded from https://msdn.microsoft.com/library/mt484311.aspx
